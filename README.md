# Projeto-Agrinho-2026
🌾 Funcionalidades do Projeto
O site do Agrinho 2026 foi desenvolvido com foco em interatividade, acessibilidade e design responsivo, utilizando HTML5, CSS3 (com variáveis nativas e Flexbox/Grid) e JavaScript Vanilla (sem bibliotecas externas).

Abaixo estão as principais funcionalidades implementadas no projeto:

📱 Menu Responsivo (Mobile First): Menu de navegação adaptável que se transforma em um botão estilo "hambúrguer" em telas menores (celulares e tablets), garantindo uma experiência de usuário fluida em qualquer dispositivo.

🖱️ Scroll Suave (Smooth Scroll): Navegação interna otimizada através de links ancorados. Ao clicar nas opções do menu, a página desliza suavemente até a seção correspondente.

💡 Cards Informativos Interativos: Na seção de Sustentabilidade, os cards utilizam eventos de clique em JavaScript para revelar informações adicionais e curiosidades ocultas, incentivando a exploração do conteúdo.

🌱 Quiz Ecológico Automatizado: Um jogo educativo dinâmico sobre o agro tecnológico. O sistema valida a resposta do usuário em tempo real, oculta a pergunta e exibe uma tela de feedback personalizado (sucesso ou correção) com a opção de reiniciar o desafio.
